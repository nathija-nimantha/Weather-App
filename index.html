About This Web Application

This web application allows users to explore weather information and country details with ease. Whether you're curious about the current weather conditions in a specific city or want to learn more about a country's population, languages, and currency, this app has you covered. Additionally, the app features a dark mode for a more comfortable viewing experience.

Key Features:

  • Weather Information: Get detailed weather forecasts including current conditions, chance of rain, wind speed, UV index, and a 7-day outlook.
  • Country Details: Search for any country and receive information on its capital, population, languages, and more.
  • Dark Mode: Switch between light and dark modes for a personalized viewing experience.
  • Interactive Map: Navigate and explore different regions around the world.
